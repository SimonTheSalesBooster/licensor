# Licensor .. Turn Your IP Into Licensing Revenue

You built a methodology, a framework, a system. Someone in another country, another format, another market could be selling it for you right now.. and sending you royalties. Licensor scans your business assets and finds the licensing deals you're leaving on the table.

## What It Does

Licensor is a Claude Code skill that scans 12 licensable asset categories across 4 dimensions and surfaces opportunities you're missing.

### The 12 Asset Categories

1. **Methodology** .. your core framework or system
2. **Certification Program** .. train others to deliver your method
3. **Community (AI Skills)** .. white-label your member tools
4. **Workshop Format** .. license your event templates
5. **YouTube Content Library** .. syndication and translation rights
6. **Podcast Methodology** .. your guest booking system
7. **Frameworks** .. proprietary models and mental maps
8. **Scorecards** .. diagnostic tools and assessments
9. **Diagnostic Tools** .. automated analysis engines
10. **AI Agent System** .. white-label your AI infrastructure
11. **Content Templates** .. proven hooks, formats, and sequences
12. **Training Materials** .. courses, decks, workbooks

### 4 Licensing Dimensions Per Asset

Every asset gets evaluated across four dimensions:

- **Territory** .. geography. Which countries or regions are untapped? Who are the natural partners there?
- **Format** .. book, app, workshop kit, SaaS tool. Can the asset live in a different container?
- **Certification** .. can institutions, universities, or corporate L&D departments license your method for internal use?
- **Technology** .. white-label AI tools, API access, embedded systems. Can your tech stack become someone else's product?

## How It Works

1. **Scans** all 12 asset categories from your business
2. **Evaluates** each across the 4 licensing dimensions
3. **Researches** potential licensees via web search
4. **Drafts** SOW-style licensing proposals with advance + royalty terms
5. **Tracks** opportunities in Partners DB and Decisions DB

## Installation

Add the skill file to your Claude Code commands directory:

```bash
cp SKILL.md ~/.claude/commands/licensor-review.md
```

Then invoke with:

```
/licensor-review
```

## Requirements

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) CLI installed
- Google Drive API access (for licensing templates)
- Notion API access (for Partners DB and Decisions DB)
- Discord webhook (for notifications)
- Web search capability (for licensee research)
